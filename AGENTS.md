  Project Overview
    A personal link in bio page built with Python and Flask where users can add URLs and the app displays them as styled link cards.
  
  Tech Stack 
    Python
	  Flask
	  HTML
	  
  File Structure
    app.py for the main application
	  Templates folder for HTML
	  Static folder for CSS
	  
  Coding Conventions
    Use clear variable names
	  Add comments for any logic that isn't obvious
	  Keep functions small and focused
	
  Testing
    After making changes, run Python app.py and confirm the app starts without errors
